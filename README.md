# Terraform-EC2
# Terraform module to create AWS EC2 instances
Amazon Elastic Compute Cloud is a part of Amazon's cloud-computing platform, 
Amazon Web Services, that allows users to rent virtual computers on which to run their own computer applications.
# Instance Type
Select the appropriate EC2 instance type based on your workload requirements, such as compute, memory, and storage needs.
# Network Settings
Configure the VPC and subnet in which your EC2 instance will reside. Additionally, set up security groups to control inbound and outbound traffic.
# Storage Options
Choose between Elastic Block Store (EBS) volumes or instance store volumes for your EC2 instanceâ€™s storage needs.
# Key Pairs
Specify an existing or create a new key pair to securely connect to your EC2 instance over SSH.
